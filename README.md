## 🚀 Live Site
[scholarsync.school](https://scholarsync.school)

---

## ✨ Features

### 🏠 Home
- Live clock (updates every second)
- Digital Student ID with barcode
- Live Myrtle Beach weather
- Dashboard stats (grades, deadlines, absences)
- Homework due today checklist
- Bell schedule with live countdown (Calin Forest HS)
- Latest announcement preview

### 📚 Academic
- Homework tracker with class tags
- Class notes organized by subject
- Deadline tracker (color-coded by urgency)
- 3D Flip Flashcards
- Pomodoro Study Timer (25/5/15 min)
- Absence tracker with HCS policy

### 🏫 School
- Monthly calendar with school events & holidays
- Admin announcements (school-wide)
- Classroom portal with grade circles & what-if calculator
- Hall Pass system with real countdown timer & pass ID
- Club portal with chat & sign-up sheets
- Sports results (W/L/T badges)
- School polls
- Lost & Found board
- Student directory with search

### 💬 Social
- Group chats (auto-delete after 2 weeks)
- Study groups with join/full badges
- Personal reminders
- Carpool board

### 👤 Profile
- Avatar color picker (8 gradient options)
- Editable display name & student ID
- Admin panel (approve teachers, post announcements, add sports results)
